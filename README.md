# 💰 Wealth - AI Finance Platform

An AI-powered personal finance management platform built with **Next.js 15**, **React 19**, **Supabase**, and **Prisma** to help users track expenses, manage budgets, and gain intelligent financial insights.

![Wealth Banner](./public/banner.jpeg)

## 🚀 Features

- 🔐 Secure Authentication & Authorization using Clerk
- 💸 Transaction Management (Income & Expenses)
- 📊 Interactive Analytics Dashboard
- 📅 Budget Planning and Expense Tracking
- 🌍 Multi-Account & Multi-Currency Support
- 🧾 Smart Receipt Scanner
- 🤖 AI-Powered Financial Insights & Recommendations
- ⚡ Background Jobs using Inngest
- 🛡️ API Protection and Rate Limiting with Arcjet
- 📱 Fully Responsive and User-Friendly Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Abhishekjadhav2004/wealth_Ai_Finanace_Finance-Platform.git
cd wealth_Ai_Finanace_Finance-Platform
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
SUPABASE_URL=
SUPABASE_ANON_KEY=
ARCJET_KEY=
```

### Run the Application

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
